# HSLink-Iso 硬件仓库

## 简述

本仓库是 HSLink-Iso 的 KiCad 硬件开源仓库，请使用 KiCad 9.0 及以上的版本打开

HSLink-Iso 为内置了隔离器的 HSLink-Pro，但是不带隔离电源输出，隔离侧需要使用目标板的电源进行供电，供电范围为 2.5V~5V

## 选型

隔离器选型如下：

- 德州仪器 ISO7741 (已验证，推荐)
- 纳芯微 NSi8241S0D (已验证)
- 川土微 CA-IS3741LB (已验证)

## 软件

HSLink Iso 软件参考<https://github.com/cherry-embedded/CherryDAP>。

## 外壳

HSLink Iso 的 PCB 适配 ulink v2 短体款外壳，请自行在电商平台选购
